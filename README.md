ES Viewer is a program created to digitize and preserve the trees made in the Europäische Stammtafeln series.

Europäische Stammtafeln is a book series created by Detlev Schwennicke. The purpose of the series is to create genealogical tables of 
the most influential families in Europe. ES contains 29 volumes in total and is an extremely useful tool for genealogists and researchers alike.

Detlev Schwennicke passed away in 2012 and ES has become an increasingly rare resource to find. The purpose of this project is to digitize and 
preserve the information in the series online so it can be accessible to everybody.

Like any series, ES has mistakes. For the purpose of this project, mistakes in the book series are not to be corrected. This is to preserve the 
original content in the series. If you wish to discuss the project, you can join my Discord server [here](https://discord.gg/9zDBhSbDmB) or you can directly send me an email [here](mailto:radistanislav63@gmail.com).

Due to the book being in German, it may be hard for some English viewers to understand. John P DuLong creates an extremely useful key [here](https://habitant.org/tools/esnotes.htm) to 
the book and its information. There is also a copy of the book’s abbreviations on the same page there.
